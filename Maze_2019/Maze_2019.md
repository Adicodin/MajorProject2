# Overview

Maze (2019, dee863ffa251717b8e56a96e2f9f0b41b09897d3c7cb2e8159fcb0ac0783611b (SHA-256)) is a Microsoft Visual C++ compiled PE32 executable. Static analysis reveals embedded strings referencing "Maze," ransom note text, hardcoded device-type identifiers (workstation, server, domain controller, etc.) used to differentiate ransom amounts per victim, references to security products like "AhnLab," and Windows Cryptographic Provider strings indicating use of the CryptoAPI for key generation and file encryption. On execution, it recursively enumerates files and directories starting from the root drive, drops a ransom note in each affected directory beginning with C:\, and writes temporary files during the encryption routine. Maze is a ransomware which operationalise double extortion — exfiltrating victim data prior to encryption and threatening public release on a dedicated leak site.
